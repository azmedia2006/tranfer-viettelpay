| 💸 | Chuyển tiền | Gửi tiền từ ViettelPay hoặc ngân hàng bất kỳ |
| 🏦 | Đa ngân hàng | Hỗ trợ: VCB, MB, ACB, MSB, TPBank, BIDV, VPBank, Techcombank... |
